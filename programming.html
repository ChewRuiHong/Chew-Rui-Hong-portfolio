Python program project

This project is to create a Mini Library Management System to store book details

This appliation comes with 5 main features:

  • Add new book
  • Update exisiting book
  • Remove exisitng book
  • View Books
  • Documentation

Details of the book includes :

  • ISBN number
  • Book title
  • Book type
  • Quantity Available
